Tutorials
=========

Homotopy theory and algebraic topology are notoriously difficult to
access without a formal, specialized. However, the actual output of
these constructions existed long before any formalism(s).

Therefore, we recommend that those unfamiliar with these formalism
concentrate on building an intuition for the input and output of these
constructions.
